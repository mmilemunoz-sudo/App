# Remis Mariela

Aplicacion moderna, minimalista y mobile-first para que Mariela registre viajes, gastos y vea resumenes diarios, semanales y mensuales.

## Pantallas incluidas

- Inicio con saludo personalizado, resumen del dia y acciones principales.
- Nuevo viaje con destino, monto, metodo de pago y notas opcionales.
- Nuevo gasto con categorias rapidas.
- Estadisticas con ganancias semanales, gastos mensuales, balance mensual y cantidad de viajes.
- Calendario mensual para ver actividad por dia.
- Historial de viajes, montos cobrados y gastos.

## Prioridad UX

- Registrar un viaje en pocos segundos.
- Botones grandes para usar entre viajes.
- Teclado numerico para montos.
- Destinos frecuentes y repeticion de recorridos.
- Interfaz liviana, tactil y clara.

## Tecnologia

- React
- Vite
- TailwindCSS
- SQLite local con `sql.js` persistido en IndexedDB

## Comandos

Cuando haya un gestor de paquetes disponible:

```bash
npm install
npm run dev
```

La app se abre en:

```txt
http://127.0.0.1:5173
```
